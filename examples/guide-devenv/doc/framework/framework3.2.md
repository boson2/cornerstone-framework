# 화면 레이아웃 잡기  #

-----------------------

- **기본적으로 12 column 레이아웃 지원**
- **화면의 폭에 따라서 4가지 크기로 구분 : 큰(lg), 중간(md), 작은(sm), 아주 작은(xs) 디바이스**

```
<body>
	<div class=“container”>
		<div class=“row”>
			<div class=“col-lg-4 col-md-5”></div>
			<div class=“col-lg-8 col-md-7”></div>
		</div>
	</div>
</body>
```
