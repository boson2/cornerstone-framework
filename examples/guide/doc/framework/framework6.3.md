# 데이터를 서버와 동기화하기  #

-----------------------

- Backbone.sync 사용
- 서버와 RESTful 한 통신을 주고 받음
- 데이터는 HTTP Body에 JSON 포맷으로 지정됨
- Model/Collection에 url 속성 지정
