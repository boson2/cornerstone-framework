# 개발 환경 구축하기  #

-----------------------

**Cornerstone CLI 설치**

- Node.js 설치 (http://www.nodejs.org/download/)
- Cornerstone CLI 설치

```
npm install -g cornerstone-cli
```

**웹 브라우저**

- WebKit 기반의 브라우저를 사용하는 것을 추천
- Chrome 14, Firefox 5, Safari 5, IE 9

**스마트 기기**

- iOS 5 이상
- Android 4.0 (ICS) 이상

**마음에 드는 에디터 구하기**

- Eclipse, WebStorm, Sublime Text, AcroEdit, Coda, …
