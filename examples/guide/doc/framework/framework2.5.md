# Cornerstone CLI #

-----------------------

**빌드**

	cstone build

- css, Javascript를 minify

**간이 웹 서버 실행**

	cstone server

- 웹 서버를 실행하고 브라우저 자동 실행
- 소스 파일 변경을 감지해서 자동 반영

**단위 테스트**

	cstone test

- mochajs, chaijs를 사용해서 테스트 케이스 작성
- PhantomJS를 사용해서 실행됨
