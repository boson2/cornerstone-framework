Cornerstone Framework 2.0 테스트

Cornerstone Framework 2.0 테스트 페이지 입니다. Unite Skin을 적용중이며, 테스트 항목 앞에 badge는 해당 테스트의 동작방식을 나타냅니다.
자동의 경우 페이지 진입 시 검증을 진행하며, 수동의 경우 사용자가 직접 눈으로 확인하여야 합니다.

  • 수동 Cornerstone UI( + Bootstrap 호환성 검증, 차트를 제외한 모든 컴포넌트, 위젯, 피쳐드 )
  • 자동 MVC
  • 자동-구현중 Mulipage Router( MVC Test에서 포함하지 못한 부분 )
  • 수동 Widget Chart
  • 자동 Util( Transition 제외 )
  • 수동 Util( Transition - 화면 전체 )
  • 수동 Util( Transition - 화면 일부분 )
  • 자동 모듈화 및 이벤트 확장
  • 수동 모듈화 및 이벤트 확장( 수동 - touch widget 등 )
